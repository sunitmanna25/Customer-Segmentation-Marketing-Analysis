# Customer Segmentation Data for Marketing Analysis

## Project Overview

This project analyses customer demographic and purchasing behaviour data to identify customer segments, buying patterns, and revenue contribution. The objective is to support data-driven marketing strategies.

## Tools and Technologies

* Python(Pandas)
* PostgreSQL
* Power BI

## Key Analysis Performed

* Customer segmentation analysis
* Revenue contribution analysis
* Purchase frequency analysis
* Age group and gender analysis
* Product category performance analysis
* Customer spending behaviour analysis

## Dashboard Features

* Interactive dashboard
* KPI tracking
* Customer segmentation visualization
* Revenue and performance insights

## Key Insights

* Older customers contribute higher revenue
* Electronics and Sports categories perform well
* Purchase behaviour varies by age and income
* Business growth opportunities identified

Kaggle Public Dataset
